## 三、核心功能
### 1. 智能工时计算
输入上下班起止时间，系统自动核算工时，计算精度可达**0.1小时**。

### 2. 薪资自动统计
支持自定义时薪，自动计算单日薪资、累计总薪资，收入数据直观展示。

### 3. 本地安全存储
所有工时、薪资数据仅保存在本地设备，无云端同步，不收集上传用户隐私数据。

### 4. 快捷搜索筛选
可通过日期、时间段、工作事项关键词检索工时记录，快速定位对应条目。

### 5. 灵活记录管理
支持单条记录编辑、删除操作，修改数据后自动重新核算工时与薪资。

### 6. 动漫风简洁界面
粉紫色动漫主题搭配毛玻璃UI设计，视觉舒适，操作流程顺滑。
